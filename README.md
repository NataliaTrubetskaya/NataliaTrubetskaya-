## Основные проекты

### 1. Анализ быстрорастущего маркетплейса

*   **Описание:** Анализ базы данных быстрорастущего маркетплейса для понимания поведения пользователей, выявления ключевых каналов роста и прибыльных категорий товаров.

*   **Задачи:**
    *   Первичный анализ и сбор данных
    *   Обзор ключевых метрик
    *   Оценка метрик монетизации и юнит-экономики
    *   Поиск инсайтов и точек роста, сегментация и формулирование гипотез
    *   Подготовка эксперимента и подведение его итогов

*   **Ключевые навыки:**
    *   A/B-тестирование
    *   Анализ юнит-экономики
    *   Поиск точек роста

*   **Артефакты:**  [Jupyter Notebook с анализом](http://localhost:8889/notebooks/Кейс%208%20.ipynb?)

### 2. A/B-тестирование онбординга в мобильном приложении для онлайн-торговли (SollmaFin)

*   **Описание:** Оценка влияния обновленного онбординга на поведение пользователей и их инвестиционную активность в международном мобильном приложении SollmaFin.

*   **Задачи:**
    *   Проведение A/B-тестирования
    *   Оценка результатов и выводы о влиянии обновленного онбординга

*   **Ключевые навыки:**
    *   A/B-тестирование
    *   Статистический анализ

*   **Артефакты:**  [Jupyter Notebook с анализом](http://localhost:8889/notebooks/Кейс%207.ipynb?)
### 3. Анализ поведения пользователей в сервисе доставки еды («Всё.из.кафе»)

*   **Описание:** Анализ пользовательского пути и поведения в сервисе доставки еды для выявления инсайтов по улучшению клиентского опыта и взаимодействия с ресторанами-партнерами.

*   **Задачи:**
    *   Анализ пользовательского поведения
    *   Поиск инсайтов для улучшения клиентского опыта и взаимодействия с партнерами

*   **Ключевые навыки:**
    *   Анализ пользовательского поведения
    *   Поиск инсайтов

### 4. Анализ юнит-экономики развлекательного приложения (Procrastinate Pro+)

*   **Описание:** Анализ юнит-экономики развлекательного приложения Procrastinate Pro+, выявление причин стагнации выручки и определение прибыльных/убыточных источников привлечения.

*   **Задачи:**
    *   Анализ юнит-экономики
    *   Поиск причин стагнации выручки
    *   Определение прибыльных и убыточных источников привлечения

*   **Ключевые навыки:**
    *   Анализ юнит-экономики
    *   Анализ каналов привлечения
    *   Поиск причинно-следственных связей

### 5. A/B-тестирование в сервисе Яндекс Книги

*   **Описание:** Проверка гипотезы в Python и составление аналитической записки на основе результатов A/B-тестирования в сервисе Яндекс Книги.

*   **Задачи:**
    *   Проверка гипотезы с использованием Python
    *   Анализ результатов A/B-тестирования
    *   Составление аналитической записки

*   **Ключевые навыки:**
    *   A/B-тестирование
    *   Python (Pandas, Scikit-learn, etc.)
    *   Статистический анализ
    *   Составление аналитических отчетов

### 6. Анализ A/B-теста в приложении с «бесконечной» лентой

*   **Описание:** Расчет параметров A/B-теста, оценка корректности проведения и анализ результатов эксперимента в развлекательном приложении с "бесконечной" лентой.

*   **Задачи:**
    *   Расчет параметров A/B-теста
    *   Оценка корректности проведения

*   **Ключевые навыки:**
    *   A/B-тестирование
    *   Статистический анализ
    *   Оценка дизайна экспериментов

### 7. Исследовательский анализ данных по заведениям общественного питания Москвы

*   **Описание:** Предобработка данных и исследовательский анализ данных по заведениям общественного питания Москвы на основе данных сервисов Яндекс Карты и Яндекс Бизнес.

### 8. Анализ поведения игроков в онлайн-игре «Секреты Темнолесья»

*   **Описание:** Анализ базы данных онлайн-игры «Секреты Темнолесья» для выявления доли платящих игроков и зависимости от расы персонажа, а также изучение процесса покупки эпических предметов.

*   **Задачи:**
    *   Определение доли платящих игроков
    *   Анализ зависимости доли платящих от расы персонажа
    *   Изучение процесса покупки эпических предметов

*   **Ключевые навыки:**
    *   Анализ пользовательского поведения
    *   Сегментация

### 9. Анализ рынка недвижимости в Санкт-Петербурге и Ленинградской области

*   **Описание:** Анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области для определения самых перспективных сегментов недвижимости, сезонных тенденций и перспективных районов.

*   **Задачи:**
    *   Определение наиболее привлекательных сегментов недвижимости
    *   Понимание сезонных тенденций на рынке
    *   Определение наиболее перспективных районов для продажи недвижимости

*   **Ключевые навыки:**
    *   Анализ рынка недвижимости
    *   Анализ данных
